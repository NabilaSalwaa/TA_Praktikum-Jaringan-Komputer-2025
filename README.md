Dokumen ini merupakan panduan praktikum jaringan komputer (CCNA Lab) dengan tujuan untuk membangun dan mengonfigurasi jaringan menggunakan router Cisco 4221 dan switch Cisco 2960, serta dua PC (PC-A dan PC-B). Praktikum ini melatih kemampuan konfigurasi IPv4 dan IPv6, pengujian konektivitas, serta penggunaan perintah dasar Cisco IOS.

ğŸ¯ Tujuan Praktikum
Menyusun topologi jaringan sesuai diagram yang diberikan.
Mengonfigurasi router dan switch dengan alamat IP, gateway, serta keamanan dasar (password, banner, enkripsi).
Mengaktifkan IPv6 routing dan memastikan komunikasi antar-subnet berjalan.
Memverifikasi konektivitas jaringan melalui perintah ping dan show di perangkat Cisco.
Menganalisis routing table dan status interface untuk memahami proses routing dan koneksi perangkat.

ğŸ§  Materi yang Dipelajari
Konfigurasi dasar router (hostname, password, IP, IPv6, banner, dan no shutdown).
Konfigurasi VLAN 1 pada switch beserta gateway.
Pengecekan status koneksi dengan show ip interface brief, show ip route, dan show ipv6 route.
Pengujian konektivitas antar-PC melalui router.
Penerapan perintah ipv6 unicast-routing untuk mengaktifkan routing IPv6.
